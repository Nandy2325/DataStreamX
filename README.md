# DataStreamX – Full Stack Music Streaming App

A feature-rich full-stack music streaming application 
with gamification, AI voice search, and personalized 
user experiences.

## ✨ Unique Features
- 🎮 Gamification System — rewards and achievements 
  for listening activity
- 🎫 Ticket Booking — book concert/event tickets 
  directly from the app
- 🎨 Customizable UI Themes — personalized visual 
  experience per user
- 🎤 Voice Search — search songs hands-free using 
  voice input
- 📚 Study Mode — distraction-free listening mode 
  for focused studying
- 👤 Avatar System — personalized user avatars 
  and profiles

## 🛠 Tech Stack
- Frontend: React.js, HTML5, CSS3, JavaScript
- Backend: Spring Boot, REST APIs, Spring Data JPA
- Database: MySQL
- Build Tool: Maven

## 📁 Project Structure
DataStreamX/
├── frontend/   → React.js UI
└── backend/    → Spring Boot REST API

## Note
Media assets (images/audio) are not included 
in this repository due to file size.
Run locally by adding your own assets 
in /src/assets folder.
